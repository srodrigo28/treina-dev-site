Mapa de Desenvolvimento Moderno

Guia completo com Next.js, TypeScript, Supabase e Zustand

1 Configuração do Projeto

Inicie o projeto com Next.js, TypeScript e Tailwind CSS usando o App Router.

npx create-next-app@latest meu-app-supabase

2 Setup no Supabase

Crie e configure seu backend no Supabase.

  1. Acesse supabase.com e crie um novo projeto.
  2. Em Project Settings > API, copie a URL e a chave anon public.

3 Integração com Next.js

Conecte o frontend ao Supabase de forma segura.

1. Instale as bibliotecas:

npm install @supabase/supabase-js @supabase/ssr

2. Crie o arquivo .env.local com suas chaves.

4 Estrutura de Pastas

Organize o projeto para escalabilidade.

  • app/: Para rotas (Home, Login, Dashboard).
  • components/: Para componentes reutilizáveis.
  • lib/: Para lógicas auxiliares (cliente Supabase).
  • store/: Para o estado global com Zustand.

5 Estado Global com Zustand

Gerencie o estado de autenticação do usuário de forma simples.

npm install zustand

6 Lógica de Autenticação

Implemente as funções de cadastro e login nos seus componentes de formulário utilizando o cliente Supabase.

7 Sincronizando o Estado

Crie um componente "ouvinte" (`AuthProvider`) que usa `supabase.auth.onAuthStateChange` para atualizar a store do Zustand sempre que o status do usuário mudar.

8 Protegendo Rotas com Middleware

Crie o arquivo middleware.ts na raiz para redirecionar usuários com base no seu status de login (protegendo a dashboard, por exemplo).

9 Dashboard e Logout

Na página do dashboard, acesse os dados do usuário a partir da store do Zustand e implemente a função de logout com `supabase.auth.signOut()`.

Fluxo Completo e Robusto

Com estes passos, você construiu uma base de autenticação moderna, segura e escalável, pronta para ser o ponto de partida de qualquer aplicação web com Next.js.
