    📊 Manual de Gráficos com Next.js

    Crie uma aplicação com 2 exemplos de cada gráfico usando Recharts e ShadCN UI.

    1. Instalação do Projeto

    # Crie projeto com Next
npx create-next-app@latest minha-aplicacao -e with-typescript

# Instale Tailwind
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# Instale ShadCN
npx shadcn-ui@latest init

# Instale Recharts
npm install recharts

    2. Configuração do Tailwind

    // tailwind.config.js
content: [
  "./app/**/*.{ts,tsx}",
  "./components/**/*.{ts,tsx}"
],
theme: {
  extend: {},
},
plugins: [],

    3. Componentes dos Gráficos

    Crie os seguintes arquivos em /components:

    • BarChartComponent.tsx e BarChartComponent2.tsx
    • LineChartComponent.tsx e LineChartComponent2.tsx
    • PieChartComponent.tsx e PieChartComponent2.tsx
    • RadarChartComponent.tsx e RadarChartComponent2.tsx
    • AreaChartComponent.tsx e AreaChartComponent2.tsx

    4. Página Principal

    Crie o arquivo app/page.tsx importando todos os gráficos:

    import { BarChartComponent } from "@/components/BarChartComponent"
import { BarChartComponent2 } from "@/components/BarChartComponent2"
// ... importe os outros

export default function Page() {
  return (
    <div className="grid grid-cols-1 md:grid-cols-2 gap-6 p-6">
      <BarChartComponent />
      <BarChartComponent2 />
      {/* os outros gráficos aqui */}
    </div>
  )
}

    5. Deploy e Finalização

    • Execute npm run dev
    • Faça o deploy em Vercel
    • Compartilhe sua página com o mundo 🚀
